# MAD Modular Apps

A zero-dependency collection of 12 modular, lightweight web application versions (Calculators, Chats, Weather Displays, and Custom HTML/CSS-only webpages) built with pure HTML/CSS and React (via CDN).

This repository is packed as a global CLI tool that lets you interactively choose and launch any of the applications locally with a single command.

## Installation

You can install it globally via npm:

```bash
npm install -g mad-modular-apps
```

Or run it directly without installation using `npx`:

```bash
npx mad-modular-apps
```

## Available Apps

1. **Calculator V1**: Standard clean calculator.
2. **Calculator V2**: Buttercream retro layout with rounded keys.
3. **Calculator V3**: High-contrast dark cyber theme.
4. **Chat V1**: Basic React messages emulation layout.
5. **Chat V2**: Dark mode green bubbles theme.
6. **Chat V3**: Corporate sky blue flat style bubbles.
7. **Weather V1**: Multi-city grid forecast guide.
8. **Weather V2**: Warm desert sand theme with lower grid layout.
9. **Weather V3**: Slate sidebar-aligned forecast layout.
10. **Webpage V1**: 3-Recipe Cookbook (Pure HTML/CSS responsive tabs).
11. **Webpage V2**: 4-Question Interactive Quiz (Pure HTML/CSS grid layout).
12. **Webpage V3**: Customer Survey Form with pop-up confirmation and resets (Pure HTML/CSS state toggles).
